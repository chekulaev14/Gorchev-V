ERP — Схема процессов

Рабочий (терминал /terminal)
Сценарий 1 — посмотреть фото детали (в течение дня)
Подходит к терминалу
Вводит PIN (4 цифры)
PIN → БД (проверка)
Выбирает категорию
Выбирает изделие
Выбирает деталь
Смотрит фото и описание
Автовыход (60 сек)
Сценарий 2 — отметить выработку (конец смены)
Подходит к терминалу
Вводит PIN
PIN → БД (проверка)
Выбирает категорию
Выбирает изделие
Выбирает деталь
Вводит количество
Видит расчёт суммы
Нажимает «Отправить»
Данные → БД (production_logs)
Автовыход (60 сек)
Таймаут бездействия — 60 секунд. Кнопка «Выход» для ручного выхода.
Кладовщик (склад /warehouse)
Открывает сайт
Вводит PIN
PIN → БД (role=warehouse)
4 таба в шапке
Номенклатура
Каталог по типам (аккордеон)
Поиск / фильтр
Клик на позицию
Карточка: фото, состав, куда входит, остаток
Остатки
Остатки по типам
Фильтр «Нулевые»
Видит что нужно докупить
Сборка
Список собираемых позиций
«Можно собрать: N шт» + нехватка
Операции
Выбор типа операции
Выбирает позицию
Вводит количество
Подтверждает
3 типа операций
Приход поставщ.
Выбирает сырьё
Кол-во + комментарий
«Оприходовать» → остаток обновлён
Приход с произв.
Выбирает деталь
Кол-во + рабочий
«Оприходовать» → остаток обновлён
Сборка
Выбирает изделие
Кол-во → предпросмотр списания
«Собрать» → компоненты списаны
Режим редактирования (кнопка «Ред.» в шапке)
Доступно
Добавить позицию
Редактировать позицию
Удалить позицию (soft)
Редактировать BOM
Доп. кнопки
Конструктор изделия (/builder)
Удалённые (/deleted) — восстановление
Директор (/warehouse, полный доступ + выработка)
Открывает сайт
Вводит PIN
PIN → БД (role=director)
5 табов (+ Выработка)
Всё то же, что у кладовщика, плюс:
Выработка
Таб «Выработка»
Сводка: кто сколько сделал, сумма
Фильтр по периоду (день/неделя/месяц)
Редактирование
Полный доступ к режиму редактирования
Конструктор, удалённые, BOM
Директор имеет полный доступ ко всем функциям склада и кладовщика.

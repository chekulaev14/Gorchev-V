# Склад — карта UI

## Навигация
URL: /warehouse
Авторизация: user login (email/password), роль WAREHOUSE или ADMIN

## Вкладки
- Номенклатура
- Остатки
- Операции
- Конструктор

## Номенклатура
- Таблица позиций: код, название, тип (изделие/заготовка/материал), ед. измерения
- Кнопка "Создать" → форма ItemForm
- Клик по строке → редактирование
- ItemForm: поля name, code, type, unit, weight, dimensions

### Сценарии
1. Создание позиции — заполнить форму, сохранить, проверить в таблице и БД
2. Редактирование — изменить поле, сохранить, проверить
3. Валидация — пустые обязательные поля, дублирование code

## Остатки
- Таблица: позиция, склад, количество
- Фильтры по складу

## Операции
- Приход: выбрать позицию, кол-во, склад → создать приход
- Сборка: выбрать изделие, кол-во → автоматическое списание компонентов

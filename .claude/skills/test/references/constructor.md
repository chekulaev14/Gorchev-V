# Конструктор изделия — карта UI

## Навигация
1. Открыть /warehouse
2. Перейти на вкладку "Конструктор" (таб в верхней части)
3. Откроется список изделий (ProductListCard)

## Список изделий
- Карточки с названием изделия, кодом, кол-вом компонентов
- Текст "BOM не настроен" если нет компонентов
- Клик на карточку → переход в редактор BOM

## Редактор BOM
Элементы:
- Кнопка "← Назад" — возврат к списку
- Название изделия + код в заголовке
- Кнопка "Сохранить" (disabled если нет изменений)
- Hint bar — подсказка текущего режима (серый = обычный, синий = режим связывания)
- ZoomControls — кнопки +/−, процент, сброс
- Колонки: "Сырьё", "Заготовки"/"Заготовка N", "Изделие"
- ItemCard — карточка компонента (название, код, тип, количество, кнопка удаления)
- AddSlot — комбобокс "+ Добавить..." для добавления позиции
- LinkOverlay — SVG-линии между карточками (bezier-кривые)

## Сценарии тестирования

### 1. Открытие конструктора
- Перейти на вкладку "Конструктор"
- Проверить: список изделий отображается
- Проверить: у каждого изделия видно кол-во компонентов

### 2. Вход в редактор BOM
- Кликнуть на изделие
- Проверить: заголовок с названием изделия
- Проверить: колонки отображаются
- Проверить: существующие связи (линии) видны

### 3. Добавление компонента
- Кликнуть на AddSlot ("+ Добавить...")
- Выбрать позицию из списка
- Проверить: карточка появилась в колонке
- Проверить: кнопка "Сохранить" стала активной (isDirty)

### 4. Создание связи (linking)
- Кликнуть на карточку в левой колонке
- Проверить: hint bar стал синим, показывает "Выбрано: ..."
- Кликнуть на карточку в правой колонке (target)
- Проверить: появилась линия между карточками
- Проверить: isDirty = true

### 5. Удаление связи
- Навести на линию между карточками
- Должен появиться крестик удаления
- Кликнуть крестик
- Проверить: линия исчезла

### 6. Сохранение
- После изменений нажать "Сохранить"
- Проверить console: нет ошибок
- Проверить БД: bom_child записи соответствуют UI

### 7. Zoom
- Кликнуть + / − на ZoomControls
- Проверить: масштаб изменился
- Проверить: линии перерисовались корректно

### 8. Отмена (Escape)
- В режиме linking нажать Escape
- Проверить: hint bar вернулся в серый режим

### 9. Unsaved changes guard
- Сделать изменение, нажать "← Назад"
- Проверить: появился confirm-диалог "Есть несохранённые изменения"

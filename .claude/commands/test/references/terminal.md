# Терминал — карта UI

## Навигация
URL: /terminal
Авторизация: worker login по PIN (4 цифры)

## Экран входа
- Поле ввода PIN
- Кнопка входа
- При неверном PIN — сообщение об ошибке

## Каталог (после входа)
- Header: имя рабочего, кнопка "Выход", toggle темы
- Секция "Изделия" — карточки категорий (изображение + название)
- Секция "Детали" — карточки заготовок
- Навигация: категория → продукты → комплектующие → деталь

### Сценарии
1. Вход по PIN — ввести PIN, проверить каталог загрузился
2. Неверный PIN — ввести неправильный, проверить ошибку
3. Навигация — пройти категория → продукт → деталь → назад
4. Выход — кнопка "Выход", проверить возврат на экран PIN
